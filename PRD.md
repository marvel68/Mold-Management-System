## 1. 产品概述

### 1.1 产品定位

面向传统模具厂的SaaS化Web管理系统，替代当前的Excel手工管理方式，提供数字化、智能化的一站式解决方案。

### 1.2 核心价值

- 📊 **实时数据可视化**：告别Excel汇总，一目了然
- 📈 **智能周透视**：自动生成供应商每周汇总报表
- 📤 **一键Excel导入**：无需改变现有工作习惯
- 📎 **附件管理**：支持PDF/图片/文档在线查看
- 🔒 **数据安全**：本地SQLite存储

### 1.3 数据来源

| 文件 | 说明 | 数据量 |
|------|------|--------|
| 2026采购日报.xlsx | 采购外协加工明细表 | 4956行×17列 |
| 2026采购日报.xlsx | 周透视表 | 54行×56列 |
| 2026采购日报.xlsx | 模具内部制作数据支撑表 | 11879行×17列 |
| 散件加工标准价-2026.01.29.xlsx | 散件标准价 | 63个Sheet |

#### 3.3.5 计算逻辑

**周总价计算：**
```
某供应商某周总价 = SUM(金额（含税）) 
  WHERE 供应商 = X 
  AND 送货日期 BETWEEN 周开始日期 AND 周结束日期
```

**示例：**
```sql
-- 计算供应商"迪米"第3周的总金额
SELECT SUM(amount_tax) 
FROM purchases 
WHERE supplier = '迪米' 
AND delivery_date BETWEEN '2026-01-12' AND '2026-01-18'
```

### 4.2 周次自动计算

```python
def calculate_week(date: date, year: int = 2026) -> str:
    """
    根据日期计算周次
    
    规则：
    - 第1周: 1.01 - 1.04 (或该年的前4天)
    - 第2周: 1.05 - 1.11
    - 以此类推...
    """
    jan_1 = date(year, 1, 1)
    days_since_jan_1 = (date - jan_1).days
    week_number = days_since_jan_1 // 7 + 1
    return f"{week_number}周"
```
